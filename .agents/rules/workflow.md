# Development Workflow

## Commands
- **Install**: `pnpm install`
- **Start**: `pnpm start` (from `site/` or root)
- **Build**: `pnpm build`
- **Deploy**: `pnpm deploy` (deploys to `gh-pages` with `baodk.tech` CNAME)

## Simple Testing
- Test components manually in the browser.
- Check mobile responsiveness (responsive views).
- Verify all links and anchor tags work.
