# Component Rules

## TypeScript Only
- All components must be `.tsx` files
- Define interfaces for props and data
- No `any` types allowed

## Icons & Assets
- Use **Material Symbols Outlined** for all icons.
- Avoid using external icon libraries unless strictly necessary.
- Image assets should be optimized for web.

## Theme Consistency
- **NO hardcoded light backgrounds** (e.g., `bg-white`).
- Use `--color-primary` and other CSS variables for interactive elements.
- Maintain consistent spacing using Tailwind's standard scales.
