# Quality Standards

## Performance
- **Benchmarks**: Aim for LCP < 2.5s and CLS < 0.1.
- Use Vite's production build optimization.
- Optimize images and use modern formats (WebP).
- Lazy load non-critical sections.

## SEO & Meta
- **Canonical URL**: `https://baodk.tech/`
- **Metadata**: Every page must have a descriptive title and meta description.
- Use semantic HTML tags (`<header>`, `<main>`, `<footer>`, `<section>`).

## Accessibility (a11y)
- Maintain contrast ratios for readability on dark background.
- Ensure keyboard focus states are visible (`outline: 2px solid var(--color-primary)`).
- Alt text required for all meaningful images.
